BoM Squad
2015
UX & UI Design

Work on Bom Squad included graphic design, user interface design and user experience design on their bill of materials online tool. For the graphic design, this included a new logo, colour palette, typography work. The user interface included all of the above, well as email template and menu reoganization - all detailed in a thirty-four page style guide. User experience work included running usability tests, as well as re-working the onboarding flow to help new customers understand the system.
